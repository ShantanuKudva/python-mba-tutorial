## How the in-browser spreadsheet works

Each lesson embeds a live, editable spreadsheet loaded from a sample workbook. You edit cells directly in the page — no downloads, no uploads.

- Click a cell and type to edit.
- Formulas starting with `=` are evaluated by x-spreadsheet.
- The **Reset** button restores the original data if you want a clean slate.
- The **Download** button exports your current sheet as a real `.xlsx` file.
