# 02 — Data with pandas (Weeks 4–5)

Two weeks where Python finally meets Excel.

| Week | Topic | Folder |
|---|---|---|
| 4 | DataFrames, reading/writing Excel, cleaning data | [`week-4/`](week-4/) |

## What you'll be able to do after these two weeks

- Read any `.xlsx` file into Python.
- Clean missing data, fix types, rename columns.
- Group, aggregate, pivot, and merge tables.
- Produce a chart and write the cleaned data back out as Excel.

This is the bread-and-butter skill set. You'll keep using it every week from here on.
