# Phase 5 — MBA Analytics (Weeks 8–11)

Four weeks. Four domains. Each week you'll:

1. Open a domain-specific dataset.
2. Apply standard MBA analytics techniques to it.
3. Produce one deliverable a real manager would actually want.

By the end of week 9, you'll have a small library of analysis patterns you'll reuse in the capstone.

## How these weeks differ from Phases 1–4

The Python is no longer the hard part — you have it. The hard part now is the **domain logic**. Read the linked references in each week's exercises index before writing code.

Typical loop:

1. Read the concept (linked in the exercises index).
2. Sketch the formula on paper.
3. Look up which pandas / numpy / scipy function to use.
4. Code it in the playground.
5. Spot-check against a known result.

If your DCF outputs $-12 trillion, something is wrong. Reasonableness-check before celebrating.

## How to study

1. Open the week's overview from the map.
2. Read the lessons in order; every code block runs in-place.
3. Hit the exercises (Easy → Medium → Hard).
4. Finish the project.
5. Click **Mark as complete** on every page.

## Done with Phase 5 when

- You can compute the standard ratios for a company's P&L + balance sheet without looking up formulas.
- You can segment customers via RFM and explain why each segment matters.
- You can forecast a 12-month series with a moving average or simple ETS, and report a confidence band.
- You can run a scenario model with a tornado chart and explain what drives outcome variance.
- You finished all four projects.
