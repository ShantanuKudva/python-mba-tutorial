# Week 2 — Collections + Control Flow

**Goal of the week:** stop using single variables. Start using **collections** of values, and write functions that operate on them.

This is where Python starts feeling like a real tool. By Friday you'll process a list of products and flag which ones to reorder — pure Python, no pandas yet.

---

## What you'll learn

- **Lists** — ordered collections (`[1, 2, 3]`).
- **Dicts** — labeled collections (`{"name": "Acme", "revenue": 1000}`).
- **Tuples** — immutable lists. Briefly.
- **Sets** — unique values. Briefly.
- `for` loops, `while` loops.
- **List comprehensions** — Python's superpower.
- **Functions** — reusable blocks of logic with inputs and outputs.

## Why it matters (MBA tie-in)

A single Excel row is a **dict**. A whole sheet is a **list of dicts**. Once you can loop and filter those, you've replicated 80% of what you do manually in spreadsheets.
