# Phase 1 — Foundations (Weeks 1–3)

Three weeks of pure Python. No pandas, no Excel libraries. Just the language itself.

Once you understand variables, loops, functions, and files, **everything else is just a library on top**. Skip this and you'll forever be cargo-culting code you don't understand.

## Map

| Week | Topic | Open |
|---|---|---|
| 1 | Setup, variables, conditionals, strings | [Week 1 overview](week-1/README.md) |
| 2 | Lists, dicts, loops, comprehensions, functions | [Week 2 overview](week-2/README.md) |
| 3 | Files, errors, modules, dates, JSON | [Week 3 overview](week-3/README.md) |

Each week has:

- **Lessons** — short concept walkthroughs. Every code block is a runnable Monaco editor — click ▶ Run.
- **Exercises** — split into 🟢 Easy, 🟡 Medium, 🔴 Hard. Each opens a fill-in-the-blanks playground.
- **Project** — small end-of-week build that ties the concepts together. Runs entirely in the browser playground.

## How to study

1. Open the week's overview from the table above.
2. Read each lesson top to bottom. Run the example code blocks.
3. Hit the exercises. Start with Easy, then Medium, then Hard.
4. Finish the week's project.
5. Click **Mark as complete** at the bottom of every page so the sidebar progress bar tracks you.

## Done with Phase 1 when

- You can read a CSV-ish text into a list of dicts, compute something useful per row, and print a summary.
- You can write a function with parameters, defaults, and a return value.
- Loops, conditionals, list/dict operations, and string formatting feel automatic.
- You finished all three weekly projects and every Easy + Medium exercise.
