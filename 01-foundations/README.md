# 01 — Foundations (Weeks 1–3)

Three weeks of pure Python. No pandas, no Excel libraries. Just the language itself.

Why? Because once you understand variables, loops, functions, and files, **everything else is just a library on top**. Skip this and you'll forever be cargo-culting code you don't understand.

| Week | Topic | Folder |
|---|---|---|
| 1 | Setup, variables, conditionals | [`week-1/`](week-1/) |
| 2 | Lists, dicts, loops, functions | [`week-2/`](week-2/) |
| 3 | Files, errors, modules | [`week-3/`](week-3/) |

Each week ends with a small CLI project that doesn't use any third-party libraries — just standard Python. By the end of week 3 you should be able to write a script that reads a CSV, computes something useful, and writes the result out.
