# 03 — MBA Analytics (Weeks 6–9)

Four weeks. Four domains. Each week you'll:

1. Open a domain-specific dataset.
2. Apply standard MBA analytics techniques to it.
3. Produce one deliverable that a real manager would actually want.

By the end of week 9, you'll have built a small library of analysis functions you'll reuse in the capstone.

## How these weeks differ from weeks 1–5

The Python is no longer the hard part — you have it. The hard part now is the **domain logic**. Read the linked references in each week's README before writing any code.

You'll often follow this loop:

1. Read the concept (linked).
2. Sketch the formula on paper.
3. Look up which pandas/numpy/scipy function to use.
4. Code it.
5. Spot-check against a known result.

That last step matters. If your DCF outputs $-12 trillion, something is wrong. Reasonableness-check before celebrating.
